# PerOpteryx
[PerOpteryx](https://sdqweb.ipd.kit.edu/wiki/PerOpteryx) is an optimization framework which improves component-based software architectures.
Quality predictions are made on the basis of models.
From 2017 until 2020 I have been involved in the extension of this framework as a student assistant.
The project is available on [GitHub](https://github.com/PalladioSimulator/Palladio-Addons-PerOpteryx).
